Project 2 — Estados de Interação CSS

Esta página demonstra o uso de pseudo-classes CSS como :hover, :focus e :active, aplicadas em componentes de interface com transições suaves e foco em acessibilidade.

Hover

Demonstração de feedback visual ao passar o mouse sobre o elemento, utilizando transições suaves.

Focus

Estilo aplicado quando o elemento recebe foco via teclado, garantindo acessibilidade.

Active

Feedback visual durante o clique, indicando ação imediata do usuário.
